## Установка и запуск

### Вариант (как в задании)

```bash
python -m venv .venv
```

Windows:

```bash
.venv\Scripts\activate
```

Linux / macOS:

```bash
source .venv/bin/activate
```

Установка зависимостей:

```bash
pip install -r requirements.txt
```

Запуск:

```bash
uvicorn main:app --reload
```



## Что открыть в браузере

- `http://127.0.0.1:8000/`
